2 Instalation and how to use

2.1 Dependencies

  • Pandoc (needs to be installed)
  • Phantom.js (needs to be installed)
  • Reveal.js (included)

2.2 Creation

  • First copy the doc folder and rename it as you like. This is not necessary but helps you organize your documents.

  • Create the md files that you want to generate. The md files are Markdown files which are nothing more than plain text files with extension md, and a lightweight markup (we should know it but it is very simple).

  • Once created the md files, we can generate html, html-slides y pdf-beamer with a script.

2.3 Build

  • To convert all md files of all the folders go to the root folder and execute:
   ./build.sh
  • To convert all md files of one folder go to the root folder and execute:
   ./build.sh [folder_name]
